# Scientific-Calculator
**Scientific Calculator:** Computer project designed to calculate values for various scientific formulas.

### How to use?
- Choose a scientific formula. Your choice can be select a number between 1-6 or X.
<img src="https://github.com/user-attachments/assets/51d89a0a-8932-418a-8148-c70c9438a027" alt="Screenshot 2024-12-13 225830" width="700">


- Select the value you would like to calculate. Your options can be a number, X or R.
<img src="https://github.com/user-attachments/assets/5d71065c-7862-43b3-b7fe-1dbe2f108112" alt="Screenshot 2024-12-13 230304" width="700">

- Enter the required values in the prompted fields.
<img src="https://github.com/user-attachments/assets/7d1743db-a4d8-464b-aa5e-e6ae64367735" alt="Screenshot 2024-12-13 230701" width="700">

- The output will be shown on the screen. Press any key to return to menu.
<img src="https://github.com/user-attachments/assets/e11e6d2e-814f-4182-a82e-370a162b3554" alt="Screenshot 2024-12-13 230908" width="700">

- After returning to the menu, you can:
  - Continue calculating another value.
  - Return to the home menu.
  - Exit the program.
